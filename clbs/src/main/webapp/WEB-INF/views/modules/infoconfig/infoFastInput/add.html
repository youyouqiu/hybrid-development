快速录入

重要！录入前请看这里

必须提前设置终端与F3平台的连接的IP，如：112.126.64.32

端口如下

通讯类型
  • 交通部JT/T808-2013
  • 交通部JT/T808-2011
  • 天禾
  • 移为
  • BDTD-SM
  • KKS
  • BSJ-A5
  • ASO
  • F3-超长待机
F3端口
  • 6975
  • 6976
  • 6978
  • 6979
  • 6981
  • 6990
  • 6991
  • 6992
  • 6993

第1 步

点击下拉框，如果发现未注册设备信息（如终端ID、终端手机号等），说明终端和平台连接设置正确，否则看左侧说明。

第2 步

确认未注册设备的终端号、终端手机号、监控对象是否正确，支持修改或新增。

相关说明

1、移为、天禾、KKS协议的终端需要手动选择或新增终端手机号和监控对象；

2、BSJ-A5、ASO和F3超待协议的终端不允许修改终端手机号；

第3 步

点击保存完成录入，终端即可上线。
